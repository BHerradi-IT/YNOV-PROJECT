## 📦 Installation

### 📋 Prérequis
- **Node.js** 18.0.0 ou supérieur
- **MariaDB** 10.6 ou supérieur
- **npm** ou yarn
- **Git** pour le versioning

### 🚀 **Démarrage Rapide**

```bash
# Cloner la repository sécurisée
git clone https://github.com/ytech-solutions-projet/YTech-Application-Web.git
cd YTech-Application-Web

# Installer les dépendances backend
cd backend
npm install

# Installer les dépendances frontend
cd ../frontend
npm install
```

### 🏃‍♂️ **Lancement des Serveurs**

```bash
# Démarrer le backend sécurisé (Terminal 1)
cd backend
node server_max_security.js

# Démarrer le frontend (Terminal 2)
cd frontend
npm start
```

### 🌐 **Accès Application Sécurisée**
- **Frontend** : http://localhost:3000
- **Backend API** : http://localhost:5000
- **Admin Dashboard** : http://localhost:3000/admin
- **Security Dashboard** : http://localhost:5000/api/security/comprehensive
- **Threat Monitoring** : http://localhost:5000/api/security/threats
- **Health Check** : http://localhost:5000/api/health
